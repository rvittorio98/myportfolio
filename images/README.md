# Images Folder

Inserisci qui le tue immagini per i progetti.

## Formati Consigliati
- JPG per foto
- PNG per immagini con trasparenza
- WebP per dimensioni ottimizzate

## Dimensioni Consigliate
- Thumbnail: 800x600px o superiore
- Gallery: 1920x1080px

## Nomenclatura
Usa nomi descrittivi senza spazi:
- progetto-nome-thumb.jpg
- progetto-nome-1.jpg
- progetto-nome-2.jpg
